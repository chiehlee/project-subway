# Project Progress Tracking

This folder contains simple progress tracking for the project-subway project.

## Structure

- `backlog/` - Tasks to be done, organized by week
- `in-progress/` - Tasks currently being worked on, organized by week
- `done/` - Completed tasks, organized by week

Each folder contains weekly files named `WN.md` (e.g., W1.md, W2.md).

## Week Numbering

- Week 1: 2026/02/02~2026/02/08
- Week 2: 2026/02/09~2026/02/15
- (Continue incrementing...)

## Usage

Move tasks between folders and weeks as progress is made. Each task should include:
- Description
- Date added/moved
- Priority (if applicable)
